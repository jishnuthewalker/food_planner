# Notion Design Analysis

This file captures the design-system notes that were in the accidental `DESIGN.md` draft so it can be kept for later reference.

## Overview

Notion presents itself as an all-in-one workspace with a confident, illustration-rich brand voice. The homepage uses a deep navy hero band, a purple pill primary CTA, and a real workspace mockup card with deep shadow. Pastel feature cards, sticky-note dots, and mesh wire illustrations are core decorative patterns.

## Key Characteristics

- Deep navy hero band with scattered sticky-note dots and mesh wire decoration
- Signature purple pill primary CTA
- Real workspace mockup embedded in the hero
- Bold yellow feature banner for high-emphasis content
- Pastel feature card palette: peach, rose, mint, lavender, sky, yellow
- Notion Sans / Inter-based typography across the surface
- Rectangular buttons and rounded cards

## Core Tokens

### Colors

- Primary: `#5645d4`
- Brand navy: `#0a1530`
- Link blue: `#0075de`
- Surface: `#f6f5f4`
- Hairline: `#e5e3df`
- Ink: `#1a1a1a`

### Typography

- Hero display: 80px / 600 / 1.05
- Heading 1: 48px / 600 / 1.15
- Body: 16px / 400 / 1.55
- Button: 14px / 500 / 1.30

### Shape

- Buttons: `8px` radius
- Cards: `12px` radius
- Pills/badges: `9999px` radius

## Components

- Primary button: purple rectangular CTA
- Secondary button: outlined rectangular CTA
- Workspace mockup card: hero embedded product UI
- Feature cards: pastel tinted content panels
- Pricing card: standard and featured variants
- Input fields: 44px text inputs with subtle borders

## Principles

- Use purple for the dominant CTA only
- Keep buttons rectangular rather than pill-shaped
- Use pastel tints for feature storytelling
- Keep shadows soft except for the hero mockup
- Preserve editorial spacing and restrained geometry
